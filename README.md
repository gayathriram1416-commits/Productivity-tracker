# Productivity Tracker

A full-stack productivity tracking web application to log work sessions, track clock-in/clock-out times, and view daily productivity reports deployed on AWS EC2 using Docker containers.

## Live Demo
Deployed on AWS EC2: `http://13.232.87.216:8080`

## Tech Stack
| Layer | Technology |
|---|---|
| Frontend | React.js, Vite, Tailwind CSS, Nginx |
| Backend | Node.js, Express.js |
| Database | PostgreSQL |
| Containers | Docker |
| Cloud | AWS EC2 (Ubuntu) |

## Features
- Start / Stop / Reset productivity timer
- Tracks clock-in and clock-out time per session
- Daily productivity report with session history
- Persistent data storage with PostgreSQL
- Fully containerized with Docker
- Deployed on AWS EC2

## Future Enhancements
- Kubernetes migration for container orchestration
- CI/CD pipeline using GitHub Actions for automated build and deploy
- End of day automated notification showing total productive hours
- Multi-user support
